What is DOS Command?

  DOS commands are a set of instructions used to interact with the operating system via a text-based interface, enabling tasks like file management, program execution, and system configuration.

  • Learn more

Who Developed MS-DOS Commands?

   Microsoft developed MS-DOS commands, led by individuals like Tim Paterson, to allow users to interact with the MS-DOS operating system for tasks such as file management and system configuration.

  • Learn more

Legacy of the Command Line

   DOS commands, integral to the Disk Operating System developed by Microsoft, were the cornerstone of early personal computing. Offering a simple yet powerful interface, they allowed users to navigate their digital worlds with precision and efficiency.

ATTRIB

Displays or changes file attributes.

BREAK

Sets or clears extended CTRL+C checking..

VER

Displays the Windows version.

TREE

Graphically displays the directory structure of a drive or path.

CHKDSK

Checks a disk and displays a status report.

CLS

Clears the screen.

COMP

Compares the contents of two files or sets of files.

COPY

Copies one or more files to another location.

DATE

Displays or sets the date.

DEL

Deletes one or more files.

DIR

Displays a list of files and subdirectories in a directory.

ECHO

Displays messages, or turns command echoing on or off.

ERASE

Deletes one or more files.

ENDLOCAL

Ends localization of environment changes in a batch file.

FIND

Searches for a text string in a file or files.

FORMAT

Formats a disk for use with Windows.

HELP

LABEL

Creates, changes, or deletes the volume label of a disk.

MKLINK

PATH

Displays or sets a search path for executable files.

PAUSE

Suspends processing of a batch file and displays a message.

PRINT

Prints a text file.

PROMPT

Changes the Windows command prompt.

PUSHD

Saves the current directory then changes it.

RECOVER

Recovers readable information from a bad or defective disk.

REM

Records comments (remarks) in batch files or CONFIG.SYS.

REPLACE

Starts a separate window to run a specified program or command.
